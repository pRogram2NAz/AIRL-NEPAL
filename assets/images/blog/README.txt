🌱 AIRL Nepal – Local Blog Article Images Directory
===================================================

Place all of your blog/article header images in this folder!

When you add or edit an article in the Admin Panel:
1. Copy their image file (e.g. "rice_fields.jpg", "drones.png") into this directory.
2. In the Admin Panel form, inside the "Image URL / Path" field, simply type the filename:
   "rice_fields.jpg"
3. The system will automatically resolve it to:
   "assets/images/blog/rice_fields.jpg"

This allows you to keep all pictures inside your own local repository!
