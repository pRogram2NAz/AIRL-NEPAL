🌱 AIRL Nepal – Local Team Member Images Directory
==================================================

Place all of your team members' profile pictures and photos in this folder!

When you add or edit a team member in the Admin Panel:
1. Copy their image file (e.g. "sita.jpg", "ram_nepal.png") into this directory.
2. In the Admin Panel form, inside the "Image URL / Path" field, simply type the filename:
   "sita.jpg"
3. The system will automatically resolve it to:
   "assets/images/people/sita.jpg"

This allows you to store and manage all pictures directly within your own local repository!
